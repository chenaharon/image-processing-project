# Results Download

Results (models, plots, visualization videos) are **not** stored in the repo (large files). To get them:

1. **Download** the Results ZIP from the link below.
2. **Extract** the ZIP into the **project root** so that you get:
   - `results_baseline/`, `results_improved/`, `results_multiclass/`, `results_deep_learning/` (per-pipeline outputs)
   - and/or `results/comparison/` (comparison outputs)

**Results download (ZIP):**

```text
https://www.dropbox.com/scl/fi/6vwhb6z9krrx8yyre7wlr/Results.zip?rlkey=k2ft86dmxfztehg649w1off4r&st=pv4363dq&dl=0
```

Use “Download” or “Direct download” on the Dropbox page, then unzip into the project root.
